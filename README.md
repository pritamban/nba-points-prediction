Project Overview
This project builds a time-series machine learning model to predict how many points Jaylen Brown will score in his next NBA game, using historical performance data and contextual features available before tip-off.

The goal is to demonstrate:
Proper framing of a forward-looking prediction problem
Careful feature engineering to avoid data leakage
Time-aware model evaluation
Clear interpretation and visualization of results
